HTML

Para este proyecto HTML se utilizó para construir la estructura básica de la aplicación, organizando el contenido de manera clara y lógica. Aquí se incluyen elementos interactivos como botones para seleccionar compuertas lógicas y tablas para mostrar los resultados. El código del aplicativo web se podría separar en 7 pasos principales que son:

1. Doctype


Establece que el documento es HTML5, especifica el conjunto de caracteres, indica a los autores, define un ícono para la pestaña del navegador, enlaza una hoja de estilo para el diseño y establece el título que se muestra en la pestaña del navegador.

2. Header


Incluye un encabezado que contiene un menú de navegación con una lista de elementos. Cada elemento de la lista tiene un enlace que lleva a diferentes secciones de la página web, como Tabla de Verdad, Objetivo, HTML, CSS, JavaScript y Canva.

3. Div Compuertas


Contiene una sección para compuertas lógicas dentro de un contenedor con la clase compuertas. Incluye un título Compuerta Lógica y un mensaje de instrucción Selecciona una opción.

Luego, presenta varios botones, cada uno con una imagen que representa diferentes compuertas lógicas: AND, OR, NOR y NAND. Cada botón tiene un atributo para identificar el tipo de compuerta y los atributos en las imágenes para describirlas.

4. Div Entradas


Contiene una sección para seleccionar el número de entradas dentro de un contenedor. Incluye un título y un mensaje de instrucción, presenta varios botones para diferentes números de entradas, cada uno con una imagen que representa la cantidad de entradas.

Los botones tienen un atributo para identificar el número de entradas correspondiente y los atributos en las imágenes para describirlas.

5. Boton


Al hacer clic en el botón, se ejecuta la función JavaScript generarTablaDeVerdad(), que se utilizó para generar la tabla de verdad en la página.

El atributo border="1" añade un borde a la tabla. También incluye una etiqueta <script src="script.js"></script> que enlaza un archivo de JavaScript externo llamado codigo.js, el cual contiene el código necesario para interactuar con la tabla y realizar la funcion de generar la tabla de verdad y mostrar la imagen de la compuerta.

7. Footer


Contiene tres secciones:

  • Creadores: Un contenedor con un título y una lista que muestra los creadores del proyecto, destacando a los estudiantes creadores del proyecto.

  • Menú de navegación: Otro contenedor con un título y una lista que contiene enlaces a diferentes secciones del sitio web, como Tabla de Verdad, Objetivo, HTML, CSS, JavaScript y Canva.

  • Logo FESC 29 años: Un enlace que abre el sitio web de la FESC en una nueva pestaña. Dentro del enlace, se incluye una imagen que muestra el logo de nuestra querida universidad.

Este pie de página proporciona información sobre los creadores, un menú de navegación y un enlace al sitio web de la institución.
